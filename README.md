Autonomous event-shortlisting app for Techment Technology. Discovers U.S. tech and Data & AI events, verifies attendee-list access, scores each against Techment's offerings, and ranks where the leadership team should show up to meet ideal prospects.

Single self-contained index.html — no build, no dependencies. Open it and press Process.

Features


One-click pipeline — the Process button runs seven agents (Discovery → Extraction → Attendee Access → Scoring → Recommendation → Persistence → Search) with live progress feedback.
Ranked shortlist — 14 events scored against techment.com, Microsoft-first (Ignite, AI Tour, FabCon lead).
Full table — S.No, Focus, Month/Date, Geo/Location, Pricing, Attendees, Website, Fit Score, Remarks. Click any row to open the event's official site.
Attendee-list tiers — ★ official list · ◐ third-party predictive (Vendelux/Lensmor) · ✕ none. 13 of 14 have accessible lists.
